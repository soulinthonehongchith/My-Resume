Packwood Ski Project, Database class

  It was a group project with a team of five. The project requested us to create a database for a ski manufacturing company, Packwood Ski. In this project, we learned to collaborate all of what we have learned from the class into the project which ranges from drawing an Entity Relation Diagram (ERD), populating data, importing data to database and querying (SQL) for data to make a report.
